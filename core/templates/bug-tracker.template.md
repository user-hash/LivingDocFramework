Each bug entry follows this format:
```

- **Status:** open | investigating | fixed
- **Priority:** P0 (critical) | P1 (high) | P2 (medium) | P3 (low)
- **Subsystem:** [affected subsystem]
- **Found:** vX.Y.Z
- **Fixed:** vX.Y.Z (if fixed)

Description of the bug.

**Root Cause:** (if known)

**Fix:** (if fixed)
```
