# Examples

## Recommended Starting Point

Start with **quickstart/** to experience the enforcement loop in 5 minutes.

## Available Examples

### Quickstart (Start Here)

A minimal runnable example with actual Python code. Experience the failure/fix loop: edit a Tier A file, get blocked, update docs, commit succeeds.

See [quickstart/README.md](quickstart/README.md)

### Doc-Systems

A testbed demonstrating doc-set structure, tier classification, and `print-context.sh` behavior. Two subsystems (api, storage) with Tier A and Tier B files.

See [doc-systems/README.md](doc-systems/README.md)

### Sample Workflow

A real-world example showing how invariants, bug patterns, and hooks connect in practice. Based on a multiplayer desync prevention scenario.

See [sample_workflow/README.md](sample_workflow/README.md)
